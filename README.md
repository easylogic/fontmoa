# Fontmoa 

Font Manager based electronjs  (built in create-react-app)

# Install 

```
git clone https://github.com/easylogic/fontmoa.git
cd fontmoa
npm install  // or yarn install
```

# Development 

Run dev server and  open electron 

```
npm run dev 

or 

yarn dev 
```
